Exercício 01 — Página de Introdução

Este exercício tem como objetivo criar uma página simples utilizando as principais tags estruturais do HTML.

Nesta página apresentamos um pequeno texto introdutório explicando o propósito do projeto e demonstrando o uso correto de elementos semânticos.

Objetivos do exercício

  • Utilizar estrutura básica do HTML
  • Aplicar títulos e parágrafos
  • Organizar conteúdo de forma semântica
  • Estruturar uma página simples
