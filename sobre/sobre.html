Sobre o projeto

Aplicação prática de conceitos essenciais de desenvolvimento front-end, com foco em consistência visual, responsividade e organização de código.

Contexto

O projeto foi criado como parte da disciplina de Programação Web, com o objetivo de colocar em prática os conceitos estudados em aula.

Cada exercício foi pensado para trabalhar uma ideia diferente, permitindo evolução gradual na construção de interfaces.

Como foi desenvolvido

As páginas foram construídas priorizando organização e clareza, buscando manter um padrão visual consistente em todo o projeto.

Também foi considerada a adaptação para diferentes tamanhos de tela, garantindo que o conteúdo funcione bem tanto no celular quanto no computador.

Objetivo

O principal objetivo foi desenvolver uma base sólida em criação de páginas web, entendendo como estruturar conteúdo e aplicar estilos de forma eficiente.
